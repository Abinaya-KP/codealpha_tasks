Overview

This repository contains the projects completed as part of my CodeAlpha Artificial Intelligence(AI) Internship. During this internship, I worked on AI and Machine Learning projects that helped me gain practical experience in Natural Language Processing, Computer Vision, and AI application development.

As part of the internship requirements, I successfully completed 3 out of 4 assigned tasks and developed working projects using Python and various AI libraries.

Completed Projects

1. Language Translation Tool

A language translation application that translates text between different languages using translation APIs. The project provides a simple and user-friendly interface for multilingual communication.

Technologies Used:

Translation API
HTML
CSS
JavaScript

2. FAQ Chatbot

An AI-powered chatbot that answers frequently asked questions related to Artificial Intelligence(AI). The chatbot uses Natural Language Processing techniques and similarity matching to provide relevant responses.

Technologies Used:

Python
NLTK
TF-IDF Vectorization
Cosine Similarity

3. Object Detection and Tracking

A computer vision project that detects and tracks multiple objects in video streams using the YOLOv8 model and OpenCV. The system identifies objects such as people, vehicles, traffic lights, and handbags while displaying bounding boxes and tracking information.

Technologies Used:

Python
OpenCV
YOLOv8
NumPy
