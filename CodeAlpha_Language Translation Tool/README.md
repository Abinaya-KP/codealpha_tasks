CodeAlpha_LanguageTranslationTool
🌍 Language Translation Tool
📌 Project Overview

This is an AI-powered Language Translation Tool developed as part of the CodeAlpha Artificial Intelligence Internship.

The application allows users to translate text between multiple languages using a translation API and provides additional features for better user experience.

✨ Features

Multi-language Translation
Copy Translation
Text-to-Speech
Swap Languages
Clear All Function
User-Friendly Interface
Responsive Design

🛠️ Technologies Used

HTML
CSS
JavaScript
Translation API

🚀 How to Use

Enter text in the input box.
Select the source language.
Select the target language.
Click the Translate button.
View the translated text.
Use Copy, Speak, Swap, or Clear features as needed.
